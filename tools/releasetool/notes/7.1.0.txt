--------------------------

Version 7.1.0

Enhancements
------------

Added directory services replication import mechanism for remote windows domain controller imports
NTDS.DIT imports now support 'dirty' databases that were not repaired after copying

Bugs Fixed
----------

NTDS.DIT import now supports Windows Server 2016 on all client platforms
jtrexe was crashing on virtualized platforms and systems without OpenCL
